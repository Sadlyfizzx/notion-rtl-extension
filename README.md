## التثبيت

حمّل الملف من الزر اللي فوق، وبعدين روح على Downloads عندك.

افتح ملف الـ ZIP، هيطلع فولدر جديد، افتحه.

بعد كده افتح Google Chrome واكتب في الشريط فوق:
chrome://extensions

فعّل Developer mode من فوق على اليمين.

بعد كده خد الفولدر اللي فتحته واسحبه جوه صفحة Chrome الي فتحناها قبل كدا.

افتح Notion وهتلاقي الكتابة العربية اتظبطت تلقائي.
